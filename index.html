Example configuration:
plugins: [
{
  xclass: 'Scaljeri.plugin.PullMenu',
  items: {
    top: {
      xclass: 'GS.view.MenuDrag', 
      mtype: 'pull',
      fill: true,
      scrollable: 'vertical', 
      id: 'my-top-menu'
    }
  }
}],
configuration options explained:
  1. xclass
    - Class name of the menu component
  3. mtype
    - The menu type
  5. fill
    - Fill the whole (parent) component (if the menu itself doesn't do so!)
  7. scrollable
    - make the menu scrollable
  9. id
    - id of the menu component

  items: {

      mtype: 'drag-append'
    },
    bottom: {
      xclass: 'GS.view.AttachedMenuHorizontal',
      mtype: 'drag-append'
    },
    left: {
      xclass: 'GS.view.AttachedMenuVertical',
      mtype: 'drag-overlay'
    },
    right: {
      xclass: 'GS.view.AttachedMenuVertical',
      mtype: 'drag-overlay'
    }	
  } 
}],
configuration options explained:
  1. xclass
    - Class name of the menu component
  3. mtype
    - The menu type: append or append-overlay
